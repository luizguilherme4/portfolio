Sobre mim

Me chamo Luiz Guilherme, tenho 22 anos e moro em Cuiabá, Mato Grosso. Desde criança eu sempre fui apaixonado por tecnologia, então decidi trilhar um caminho no ramo da programação. Sou um Desenvolvedor Front-End com conhecimento em React, Nextjs, Git/GitHub, Nodejs, MySQL, JavaScript, TypeScript, HTML, CSS e Figma.

Atualmente estou me aprofundando na área de desenvolvimento back-end, estudando especificamente Nodejs, para futuramente me tornar um desenvolvedor full-stack e poder participar de grandes projetos de ponta a ponta, esse é meu objetivo. Como um grande fascinado por desenvolvimento de software, tenho muito interesse em aprender e me qualificar em novas tecnologias.

A tecnologia é a minha maneira de mudar o mundo e é por isso que amo o que eu faço!

Projetos

Staart Auth

Esse projeto é uma demonstração de aplicação que utiliza o Firebase para fazer Login e Cadastro de usuários de uma plataforma. O projeto foi feito em React e algumas das features incluem a possibilidade de trocar o e-mail de cadastro e redefinir a senha.

GitHub Site

PokeNext

PokeNext é uma pokédex que apresenta os dados de mais de 250 Pokémons. O projeto foi desenvolvido em Next.js e consome a api PokéApi e as imagens foram retiradas do repositório de sprites da PokéApi.

GitHub Site

AluraTube

AluraTube é um projeto desenvolvido em Nextjs e estilizado com Styled Components que tem como propósito ser ter uma UI clone do Youtube e suas principais features são o Dark Mode e a barra de pesquisa de vídeos.

GitHub Site

DizAí

DizAí é um projeto desenvolvido em React que permite que os usuários criem posts contando histórias.

GitHub Site

Mentalista

Mentalista é um jogo de adivinhação em que um número aleatório é sorteado e o usuário possui quatro chances de acertar. Durante as tentativas do usuário, dicas aparecem na tela indicando se o número digitado é maior ou menor que o número sorteado.
